简介

在一台临时的电脑上想快速体验 MC ？打开 MCJS.CC ，不用下载 Java/启动器/客户端，不需要繁琐的配置操作，打开网页就能玩！
Eaglercraft 介绍：Eaglercraft 是在 Javascript 运行的网页版 MC，基于 MCJE。使用 TeaVM 和 OpenGL 模拟器运行浏览器兼容的 Java 虚拟机。
MCJS 介绍：专为简体中文用户优化的 Eaglercraft 中文版，汉化最完整，还原原版体验。
JS 版 vs WASM 版：JS 版兼容性更广；WASM 版性能更强、帧率更高，建议电脑 Chrome 优先使用。

常见问题（FAQ）

问：游戏打不开/卡死/崩溃

答：请使用最新 Chrome，优先用 WASM 版。设备性能越好越流畅。

问：多人联机问题

答：支持局域网与 P2P 直连。需同中继服务器、公网IP或NAT1。

问：手机/平板支持

答：仅 MCJS 1.8.8 适配触屏，其他版本不保证。

问：1.13+ / 模组支持

答：不支持，技术上限为 1.12.2。

问：离线下载

答：暂不开放，未来可能恢复。
